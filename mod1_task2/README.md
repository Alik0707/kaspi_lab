В данной работе реализованы классы Cart и Order для описания корзины заказов.
Класс Order содержит информацию о наименовании товара, категории и стоимости.
Класс Cart хранит заказы в коллекции List, реализованной с помощью ArrayList.

Добавление заказов выполняется через метод addOrder, использующий метод add() коллекции ArrayList.

Поиск заказов с ценой выше заданного значения реализован путём предварительной сортировки списка заказов по убыванию стоимости и последующего линейного обхода. Обход прекращается при обнаружении заказа с ценой ниже указанного значения, что снижает количество операций.

Для подсчёта количества заказов в каждой категории используется коллекция Map, где ключом является категория, а значением — количество заказов.
